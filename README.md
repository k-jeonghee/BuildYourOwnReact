## Didact - 나만의 리액트 만들기
VanillaJS로 React 구현하기

## 프로젝트 목표
1. React의 렌더링 구조를 학습합니다.
2. 제공된 코드 외 React Hook을 구현하여 적용합니다.
3. JSX를 이용하지 않고 React와 동일하게 동작하도록 구현합니다.

## 추가 구현 사항
1. React.memo
2. useEffect | useCallback | useMemo
